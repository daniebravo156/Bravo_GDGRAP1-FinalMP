# Bravo_GDGRAP1-FinalMP
Members: Danielle R. Bravo 
Github Link: https://github.com/daniebravo156/Bravo_GDGRAP1-FinalMP.git 

Controls: 
Mouse move camera in 3rd person
W/S move tank forward and backward in 3rd person
A/D turn tank left and right in 3rd person
1 switch between 3rd person and binocular mode
2 switch to top view
W/A/S/D pan in top view
W/A/S/D look around in binocular mode
Q/E zoom binocular in and out
F cycle point light intensity
Tab toggle cursor
Esc close the game
